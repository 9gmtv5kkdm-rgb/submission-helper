# Submission Helper / 投稿辅助工具

> 学术投稿全环节实战工具箱 —— 从 Cover Letter 到 Rebuttal，一站式搞定。

## 这是什么？

`submission-helper` 是一个学术论文投稿辅助技能。它不帮您选期刊（那是 journal-recommender 等工具的活），它帮您**把论文投出去并应对审稿流程**。

## What is this?

Submission Helper is an academic manuscript submission toolkit. It doesn't help you find journals — it helps you **submit your manuscript and navigate the peer review process**.

## 与已有技能的区别 / Difference from Existing Skills

| 已有技能 | 功能 | 本技能补充 |
|----------|------|-----------|
| journal-recommender | 推荐期刊 | 写完 Cover Letter 投出去 |
| journal-submission | 期刊调研 | 逐条检查投稿材料是否齐全 |
| journal-matchmaker | 匹配期刊 | 收到审稿意见后怎么写 Rebuttal |
| resubmission-deadline-tracker | 跟踪截止日 | 审稿超时怎么催稿、怎么撤稿 |

## 快速开始 / Quick Start

```
# 准备投稿
1. 选择 Cover Letter 模板 → 替换占位符 → 完成
2. 逐条核对投稿检查清单 → 补全材料 → 投出

# 收到审稿意见
3. 阅读审稿信 → 分类审稿意见 → 确定优先级
4. 按 Rebuttal 框架逐条回复 → 标注修改位置 → 提交修改稿

# 等待期间
5. 跟踪投稿状态 → 超时发催稿邮件 → 如需撤稿用撤稿模板
```
